# 🖱️ hyprcursor-sync  
> Because manually updating cursors is for NPCs.

## 🚀 What is this?
`hyprcursor-sync` is a **simple auto cursor updater** for **Hyprland (only)** users who are tired of:
- mismatched cursors 😐  
- broken themes 💀  
- “why didn’t my cursor update??” moments  

This script **syncs and updates your cursor properly**, so Hyprland stops gaslighting you.



## 😤 The Problem
You:
- change cursor theme  
- restart Hyprland  
- still see the old cursor  

Hyprland:
> *“Works on my machine.”*

Yeah… no.



## 🧠 The Solution
This script:
- Updates your cursor config  
- Syncs it correctly with Hyprcursor  
- Makes sure your cursor **actually changes**  
- Saves you from ritual sacrifices and forum-diving  
- (Cursor might take time to reload if youre on a low-end system... ok?)


## ✨ Features
- 🔄 Auto-sync cursor changes  
- ⚡ Lightweight & fast  
- 🧼 No bloat, no BS  
- 🐧 Linux-only, as it should be  



## 📦 Requirements
- Hyprland  (Obviously)
- Hyprcursor  
- NWG-Look
- A brain (optional, script does the thinking) 



## 🛠️ Installation
